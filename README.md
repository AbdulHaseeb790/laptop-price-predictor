# 💻 Laptop Price Predictor

A Machine Learning web app that predicts laptop prices based on specifications like RAM, CPU, storage, and more.

---

## 📌 Features

- Predict laptop price instantly
- Interactive web interface using Streamlit
- Supports multiple laptop configurations
- Uses real-world dataset for training
- Clean and simple UI

---

## 🧠 Machine Learning

- Model Used: Random Forest Regressor
- Preprocessing:
  - OneHotEncoding for categorical features
  - Feature engineering (PPI calculation)
- Evaluation Metrics:
  - R² Score
  - Mean Absolute Error (MAE)

---

## 📊 Input Features

- Brand
- Type
- RAM (GB)
- Weight (kg)
- Touchscreen
- IPS Display
- Screen Size
- Screen Resolution
- CPU
- HDD
- SSD
- GPU
- Operating System

---

## ⚠️ Limitations

- Model is trained on older laptop dataset
- May not accurately predict prices for:
  - Latest generation CPUs (e.g., i5 12th/13th gen)
  - Refurbished or upgraded laptops (e.g., 32GB RAM on old machines)

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
